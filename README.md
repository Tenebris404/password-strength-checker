# Password Strength Checker

This is a simple Python project that checks how strong a password is based on basic security rules.



## Features
- Checks password strength using multiple rules
- Detects commonly used passwords
- Provides improvement suggestions
- Estimates password crack time (basic)
- Option to save results to a report file


## Why This Project
Weak passwords are one of the most common security issues.
This project helps users understand how to create stronger
and safer passwords using basic security principles.
